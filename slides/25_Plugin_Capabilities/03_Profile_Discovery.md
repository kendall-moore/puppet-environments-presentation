## Profile Discovery

* Need to know what profiles exist before scanning
* Mines the XCCDF file for a list of supported profiles
* Returns the list from all Nodes

<br/>

```shell
$ mco oscap profiles
```

```shell
master
   OpenSCAP Profiles: ["rht-ccp"]

Finished processing 1 / 1 hosts in 172.97 ms
```
